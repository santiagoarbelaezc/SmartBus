<h1 align="center">🚌 SmartBus — Android App</h1>
<h3 align="center">Aplicación nativa Android para compra de tiquetes y rastreo de buses en tiempo real</h3>

---

## 📌 Descripción

Módulo Android de **SmartBus**, desarrollado en **Kotlin + Jetpack Compose**. Permite buscar rutas, comprar tiquetes, rastrear el bus en tiempo real y acumular puntos de fidelización, todo desde el celular.

Arquitectura **Clean Architecture + MVVM**, conectado a una API REST centralizada desplegada en AWS.

---

## 📱 Pantallas

| Pantalla | Descripción |
|----------|-------------|
| Login / Register | Autenticación de usuario |
| Home | Panel principal con accesos rápidos |
| Búsqueda | Filtrado por origen, destino y fecha |
| Selección de Asiento | Mapa interactivo del bus |
| Pago | Integración con pasarela certificada |
| Mis Tiquetes | Activos e historial con código QR |
| Rastreo GPS | Bus en tiempo real sobre mapa |
| Puntos y Niveles | Bronce, Plata y Oro |
| Crédito de Viaje | Viaja ahora, paga después |
| Perfil | Datos del usuario y configuración |

## 🚀 Instalación
```bash
git clone https://github.com/tu-usuario/smartbus-android.git

# Agregar en /app/
# - google-services.json

# Agregar en local.properties
MAPS_API_KEY=your_google_maps_key
BASE_URL=https://api.smartbus.com/v1/

# Compilar
./gradlew assembleDebug
```
